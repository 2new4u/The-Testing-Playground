# The Testing Playground

A simple web application designed as a sandbox to practice automation tools like Playwright, Selenium and Cypress.
A complete and ready-to-use version of this project can be found at https://2new4u.net/playground

## Getting Started

### 1. Clone the repository
```bash
git clone [[https://github.com/your-username/qa-sandbox.git](https://github.com/your-username/qa-sandbox.git)](https://github.com/2new4u/The-Testing-Playground.git)
```

### 2. Install Dependencies
Ensure you have Python installed, then run:
```bash
pip install flask
```

### 3. Run the Sandbox
```bash
python playground.py
```
